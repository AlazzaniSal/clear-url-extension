# رابط واضح / Clear URL

إضافة كروم لنسخ الروابط بشكل واضح ومقروء بدل ظهورها بصيغة مشفّرة طويلة.

</p>

## المميزات

- نسخ الروابط العربية ومتعددة اللغات بشكل مقروء.
- إزالة باراميترات التتبع مثل `utm_source` و `fbclid` و `gclid`.
- دعم قائمة الكليك يمين لنسخ رابط الصفحة أو أي رابط داخل الصفحة.
- خيار تغيير اللغة داخل الإضافة: تلقائي، إنجليزي، عربي، صيني.
- تعمل محليًا داخل المتصفح.

## التثبيت المحلي

1. افتح `chrome://extensions`.
2. فعّل **Developer mode**.
3. اضغط **Load unpacked**.
4. اختر مجلد الإضافة.
