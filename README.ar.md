## المميزات

- نسخ الروابط العربية ومتعددة اللغات بشكل مقروء.
- إزالة باراميترات التتبع الشائعة مثل `utm_source` و `fbclid` و `gclid`.
- دعم قائمة النقر بالزر الأيمن لنسخ رابط الصفحة أو أي رابط داخل الصفحة.
- مبدل لغة مدمج: تلقائي، إنجليزي، عربي، صيني، ألماني، فرنسي، ياباني، روسي، برتغالي، وإسباني.
- تعمل محليًا داخل المتصفح.

## التثبيت المحلي

1. افتح `chrome://extensions`.
2. فعّل **Developer mode**.
3. اضغط **Load unpacked**.
4. اختر مجلد الإضافة.
