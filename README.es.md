# URL clara / Clear URL

Una extensión de Chrome para copiar URL en un formato claro y legible en lugar de texto codificado largo.

## Funciones

- Copia URL multilingües en un formato legible.
- Elimina parámetros comunes de seguimiento como `utm_source`, `fbclid` y `gclid`.
- Incluye soporte para el menú de clic derecho en URL de páginas y enlaces.
- Selector de idioma integrado: automático, inglés, árabe, chino, alemán, francés, japonés, ruso, portugués y español.
- Funciona localmente en el navegador.

## Instalación local

1. Abre `chrome://extensions`.
2. Activa **Developer mode**.
3. Haz clic en **Load unpacked**.
4. Selecciona esta carpeta de la extensión.
