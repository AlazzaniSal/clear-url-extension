# クリアURL / Clear URL

長いエンコード済みテキストではなく、読みやすい形式でURLをコピーするためのChrome拡張機能です。

## 機能

- 多言語URLを読みやすい形式でコピーします。
- `utm_source`、`fbclid`、`gclid` などの一般的な追跡パラメータを削除します。
- ページURLやリンクをコピーする右クリックメニューに対応しています。
- 内蔵の言語切り替え：自動、英語、アラビア語、中国語、ドイツ語、フランス語、日本語、ロシア語、ポルトガル語、スペイン語。
- ブラウザ内でローカルに動作します。

## ローカルインストール

1. `chrome://extensions` を開きます。
2. **Developer mode** を有効にします。
3. **Load unpacked** をクリックします。
4. この拡張機能フォルダを選択します。
