# 清晰链接 / Clear URL

一个 Chrome 扩展，用于以清晰、可读的格式复制链接，而不是复制很长的编码文本。

<p align="center">
  🇺🇸 <a href="README.md">English</a> |
  🇸🇦 <a href="README.ar.md">العربية</a> |
  🇨🇳 <a href="README.zh.md">中文</a>
</p>

## 功能

- 以可读格式复制多语言链接。
- 移除常见跟踪参数，例如 `utm_source`、`fbclid` 和 `gclid`。
- 支持右键菜单复制页面链接或页面内链接。
- 内置语言切换：自动、英语、阿拉伯语、中文。
- 在浏览器本地运行。

## 本地安装

1. 打开 `chrome://extensions`。
2. 启用 **Developer mode**。
3. 点击 **Load unpacked**。
4. 选择扩展文件夹。
