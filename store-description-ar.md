# رابط واضح - وصف متجر Chrome

انسخ الروابط العربية والروابط متعددة اللغات بشكل واضح ومقروء بدل ظهورها بصيغة طويلة مثل %D9% و %D8%.

## المميزات
- نسخ الرابط الحالي بشكل واضح ومقروء.
- نسخ الرابط الأصلي المشفّر عند الحاجة.
- إزالة باراميترات التتبع مثل utm و fbclid و gclid.
- خيار كليك يمين لنسخ رابط الصفحة أو أي رابط داخل الصفحة.
- يعمل محليًا داخل المتصفح بدون إرسال بياناتك لأي خادم.

## مناسب لمن؟
- صناع المحتوى.
- المسوقين.
- أصحاب المتاجر.
- أي شخص يشارك روابط عربية أو روابط بلغات غير إنجليزية.
