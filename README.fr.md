## Fonctionnalités

- Copie les URL multilingues dans un format lisible.
- Supprime les paramètres de suivi courants comme `utm_source`, `fbclid` et `gclid`.
- Prend en charge le menu clic droit pour les URL de pages et les liens.
- Sélecteur de langue intégré : auto, anglais, arabe, chinois, allemand, français, japonais, russe, portugais et espagnol.
- Fonctionne localement dans le navigateur.

## Installation locale

1. Ouvrez `chrome://extensions`.
2. Activez **Developer mode**.
3. Cliquez sur **Load unpacked**.
4. Sélectionnez ce dossier d'extension.
