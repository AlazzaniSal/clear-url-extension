Uma extensão do Chrome para copiar URLs em um formato claro e legível em vez de texto codificado longo.

## Recursos

- Copia URLs multilíngues em um formato legível.
- Remove parâmetros comuns de rastreamento, como `utm_source`, `fbclid` e `gclid`.
- Suporta menu de clique direito para URLs de páginas e links.
- Seletor de idioma integrado: automático, inglês, árabe, chinês, alemão, francês, japonês, russo, português e espanhol.
- Funciona localmente no navegador.

## Instalação local

1. Abra `chrome://extensions`.
2. Ative **Developer mode**.
3. Clique em **Load unpacked**.
4. Selecione esta pasta da extensão.
