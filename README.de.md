## Funktionen

- Kopiert mehrsprachige URLs in einem lesbaren Format.
- Entfernt gängige Tracking-Parameter wie `utm_source`, `fbclid` und `gclid`.
- Unterstützt das Rechtsklick-Menü für Seiten-URLs und Links.
- Integrierter Sprachwechsler: Auto, Englisch, Arabisch, Chinesisch, Deutsch, Französisch, Japanisch, Russisch, Portugiesisch und Spanisch.
- Läuft lokal im Browser.

## Lokal installieren

1. Öffnen Sie `chrome://extensions`.
2. Aktivieren Sie **Developer mode**.
3. Klicken Sie auf **Load unpacked**.
4. Wählen Sie diesen Erweiterungsordner aus.
