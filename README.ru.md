# Clear URL / Читаемый URL

## Возможности

- Копирует многоязычные URL в читаемом формате.
- Удаляет распространенные параметры отслеживания, например `utm_source`, `fbclid` и `gclid`.
- Поддерживает меню правого клика для URL страниц и ссылок.
- Встроенный переключатель языков: авто, английский, арабский, китайский, немецкий, французский, японский, русский, португальский и испанский.
- Работает локально в браузере.

## Локальная установка

1. Откройте `chrome://extensions`.
2. Включите **Developer mode**.
3. Нажмите **Load unpacked**.
4. Выберите папку этого расширения.
